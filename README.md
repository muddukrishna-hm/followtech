# followtech

A curated list of blogs, researchers, newsletters, and resources for staying current on software engineering, distributed systems, and data-intensive applications.

## Structure

| Folder | Description |
|--------|-------------|
| [books/](books/) | Technical books and companion sites |
| [researchers/](researchers/) | Individual researchers and engineers |
| [engineering-blogs/](engineering-blogs/) | Company engineering blogs |
| [newsletters/](newsletters/) | Email newsletters and digests |
| [research-papers/](research-papers/) | Paper summaries and academic outlets |
| [architecture/](architecture/) | Scalability, systems design, and architecture |

## Quick links

- [All sources (flat list)](SOURCES.md)

## How to use

1. Browse by category folder.
2. Each entry has a dedicated markdown file with URL, focus area, and optional notes.
3. Star or watch sources you want to follow; add your own notes in each file.
